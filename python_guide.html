Python Guide

This is a simple document to walk you through the installation of python and ipython notebook (one of the Integrated Development Environment IDE).

Python 2 or Python 3?

In most of the case, no matter you are using mac, windows, or ubantu, one or more versions of python is pre-installed. Currently, there are two prevailing versions of python: python 2.X and 3.X. Python 3 is slightly newer but, python 2 could be more stable and have more library support. Generally, python 2 is recommended for its stability and package avaialbility. And I, the GSI, am more familiar with python 2 than python 3.

The syntax of them are almost identical with only a few exceptions. A comparison between them can be found here. Most packages that are going to be used in this class, such as sklearn, are supported by both python 2 and python 3. As mentioned in class, we will try to accommodate both versions.

Coding in Python

Generally, there are two ways of coding python

  1. Use IDE : recommended for beginners.
    IDEs allow you to write script and run it, see the result simultaneously! ipython notebook is one of the leading IDE for python. I personally like it most comparing to several IDEs including pycharm and spider because of the simplicity of use and clarity of the outputs. You could explore more about ipython notebook here here.

  2. Use text editors : the expert way.
    Code in any text editors and use bash shell to run python script. There are some good choises of text universal editors, such as Atom and Sublime. Check this article out about on how to run your python scripts. Text editors are smoother in text editing due to their light weights and highly customizable for different languages. This approach is also useful when you need to run something remotely on the server where only a shell (but no GUI) is available to you.

Some personal experience: I use text editors to write python scripts. I will run the scripts using ipython notebook if I would like to store and document the outputs, especially the plots.

Installation of python and ipython notebook

No matter if you have a python or not in your computer, you can install python and ipython notebook, as well as other dependencies and packages through Anaconda.

The installation is pretty easy on all operating systems, check this page out: http://docs.continuum.io/anaconda/install

After having it installed, you are ready to run python through ipython notebook. Just start a bash shell in terminal (mac) or a cmd window (windows). Type:

$ ipython notebook

If you see a new tab appearing in your browser, you are all set! If not, paste the error message to google and search how other people solved it. If no satisfactory answer is found, paste your error message and a screenshot to Piazza. We will help you!

If you cannot wait to play with python using ipython notebook, follow this page. We will go through more examples later in class.
